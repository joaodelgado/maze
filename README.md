# Circle packing

Small [Piston](https://github.com/PistonDevelopers/piston) project to visualize
maze generation.

![screenshot](./screenshot.png)

The following algorithms are supported:

  - Depth first search
  - Randomized Kruskal
  - Randomized Prim

**How to run**

```
cargo run --release -- -g [dfs|kruskal|prim]
```
